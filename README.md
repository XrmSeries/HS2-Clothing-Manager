`HS2 Clothing Manager` 将 Maker 中按分类浏览资源的体验扩展到 Studio、本体场景和 H 场景。无需为了更换一件上衣、一双鞋或一件内衣反复加载服装卡：选中角色，从完整资源库中找到物品，直接应用到对应槽位。

插件读取原版资源与 Sideloader 已注册的 zipmod 内容，为拥有大型服装 Mod 收藏的用户提供统一、可检索、可管理的工作流。

## ✨ 核心能力

- **单槽位直接换装**：不替换整套 Coordinate，只修改当前需要的部件。
- **完整资源浏览器**：原版与 zipmod 资源按类别统一显示，支持缩略图、分页和预加载缓存。
- **快速筛选**：支持关键词搜索、原版/Mod 筛选、收藏筛选、隐藏项筛选和自定义分组。
- **当前穿戴定位**：显示角色当前穿戴的物品，可快速跳回资源库中的对应位置。
- **Mod 来源追踪**：可显示物品的 zipmod、GUID 和槽位信息，便于排查、整理与定位资源。
- **收藏与整理**：将常用物品加入收藏，隐藏不需要的条目，或建立自己的服装主题分组。
- **安全的 UI 输入处理**：窗口获得鼠标焦点时阻止相机旋转和场景误操作。
- **多语言界面**：内置中文、英文和日文界面选项。

## 👚 支持的资源类型

### 🧥 服装

可独立替换以下服装槽位：

- 上衣
- 下衣
- 胸罩/内衣上装
- 内裤/内衣下装
- 手套
- 连裤袜
- 袜子
- 鞋子

### 💄 外观部件

- 前发、后发、侧发、发型配件
- 头型、面部皮肤、面部细节
- 眉毛、眼睛、眼影、睫毛、腮红、唇妆、泪痣
- 身体皮肤、身体细节、晒痕、乳头、体毛、花纹
- 头部、耳部、眼镜、面部、颈部、肩部、胸部、腰部、背部、手臂、手部、腿部和胯部饰品

饰品应用时会优先替换同类型的现有槽位；没有同类型物品时，会尝试使用空闲槽位。

## 🎬 场景工作流

### 🛠️ Maker

进入 Maker 后，顶部中间会出现收缩入口与渐隐提示。展开窗口后即可浏览资源、查看当前穿戴、搜索、收藏或直接替换单个槽位。Maker 原生列表会在适用时同步到新选择的物品。

### 🏛️ Studio

Studio 是本插件的重要使用场景。

1. 在 Studio 树状列表中选中一个或多个角色。
2. 场景进入时会显示顶部中间收缩入口，点击箭头展开。
3. 选择类别和物品后，直接应用到全部已选角色。

这让 Studio 中的角色换装不再依赖服装卡。需要为多名角色统一换鞋、内衣、发型或服装部件时，可在一次操作中完成。

### 🎮 本体与 H 场景

插件也会在普通游戏场景和 H 场景提供入口。

- 场景加载时自动显示顶部中间收缩条，不遮挡游戏界面。
- 默认优先选择当前加载的女性角色。
- 当场景中存在多个角色时，可在窗口顶部切换目标。
- `Ctrl + F9` 可随时显示或隐藏窗口。

## 🗂️ 资源管理

面对数千个 zipmod 条目时，核心不是“更多列表”，而是更快找到真正需要的物品。

| 功能 | 用途 |
| --- | --- |
| 缩略图浏览 | 快速确认外观，无需逐项试穿 |
| 搜索 | 支持按名称、GUID 或 zipmod 名称查找 |
| 收藏 | 建立常用服装与外观部件库 |
| 隐藏 | 从日常浏览中移除重复、过时或不需要的资源 |
| 自定义分组 | 创建角色主题、场景主题或拍摄主题资源集 |
| 当前穿戴 | 立即识别角色正在使用的资源 |
| zipmod 定位 | 追溯资源来源并定位对应 Mod 文件 |

EX 版本还支持从某一个 zipmod 入口直接浏览该 Mod 所提供的全部兼容物品。

## 🔥 H 场景处理

H 场景中的单槽位换装需要考虑角色当前的脱衣与视觉状态。

### 💎 EX

EX 会在服装刷新前记录角色状态，并在新服装加载完成后恢复兼容状态。只要目标服装支持原有状态，插件会尽量维持：

- 半脱与全脱服装状态
- 湿身状态
- 汁液等级
- 乳头状态
- 光泽状态
- 其他兼容的服装槽位状态

这样可以减少“换一件衣服导致当前场景状态被打断”的问题。

### 🧩 SE

SE 只执行基础换装。H 场景中替换服装后，被更换槽位会恢复为正常穿戴状态；SE 不记录、不复制、不恢复半脱、全脱、湿身、汁液、乳头、光泽或其他服装槽位状态。

发生此类换装时，SE 会显示顶部中间的中文渐隐提示，明确说明该限制。

## 🚀 安装与使用

### 🔗 前置依赖

- BepInEx 5
- HS2API
- Sideloader
- ExtensibleSaveFormat

### 📦 安装步骤

1. 下载并选择 **SE** 或 **EX** 其中一个 DLL。
2. 放入游戏目录的 `BepInEx/plugins`。
3. 启动 Maker、Studio 或游戏，本插件配置文件会自动生成。
4. 使用 `Ctrl + F9` 切换窗口，或在场景加载后的顶部中间收缩条中打开窗口。

### ⚙️ 配置项

可在 BepInEx Configuration Manager 中调整：

- 界面语言
- 每页显示数量
- 非焦点窗口透明度
- 窗口宽度与高度
- 打开/关闭窗口的快捷键

## 🛡️ 兼容性与注意事项

- 插件会读取 Sideloader 已成功注册的内容；损坏、缺失或不兼容的 zipmod 仍可能由游戏本身报出缺失脚本、材质或 Renderer 警告。
- MoreAccessories、StudioAccessoryNames 等插件的饰品槽报错属于它们自身的兼容链路；普通服装槽位换装不依赖这些插件。
- 换装功能旨在修改角色当前外观数据，不替代服装卡管理、场景管理或完整角色编辑器。
- H 场景中建议优先确认目标角色与目标服装兼容性，特别是包含复杂骨骼、布料或特殊 Shader 的 Mod。
