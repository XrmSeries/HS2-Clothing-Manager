## ✨ Features

HS2 Clothing Manager is a full-featured clothing management plugin for HoneySelect2, providing an intuitive visual interface to effortlessly browse, search, favorite, wear, and manage your entire clothing collection.

### 📦 Core Features

| Feature | Description |
|:---|:---|
| 🗂️ **Category Browsing** | Browse by 5 major categories — Clothing, Hair, Face, Body, Accessories — each with full sub-categories |
| 🖼️ **Thumbnail Preview** | Grid layout with thumbnails for quick visual browsing |
| ⭐ **Favorites** | One-click favorite any item, filter to access them instantly |
| 👁️ **Hide Items** | Hide unwanted items to keep your list clean and focused |
| 📁 **Custom Groups** | Create custom groups to organize items your way, with cross-category mixing |
| 🔍 **Smart Search** | Search by name, GUID, or Zipmod name — press Enter to refresh |
| 👗 **Quick Wear** | Wear items directly from the plugin in the character maker |
| 🎯 **Current Outfit** | Real-time display of the currently worn item in the selected category |
| 📋 **Copy Info** | One-click copy of item details to clipboard |

### 🎛️ Customization

| Setting | Description |
|:---|:---|
| 🌐 **Multi-Language** | Supports 中文, English, 日本語 — changes apply instantly without restart |
| 📄 **Page Size** | Customize items per page (10–60) |
| 🔅 **Window Opacity** | Adjust window opacity when unfocused (20%–100%) |
| 📐 **Window Size** | Freely adjust window width and height |

### ⌨️ Shortcuts

- **CTRL + F9** — Show/hide the plugin window

## 📥 Installation

1. Make sure [BepInEx 5.x](https://github.com/BepInEx/BepInEx), [KKAPI](https://github.com/IllusionMods/KKAPI), and [Sideloader](https://github.com/IllusionMods/Sideloader) are installed
2. Place `HS2_ClothingManager.dll` into the `BepInEx/plugins/` directory
3. Launch the game — the plugin loads automatically when entering the character maker
