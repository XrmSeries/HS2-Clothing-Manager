## ✨ 功能介绍

HS2 Clothing Manager 是一款为 HoneySelect2 设计的服饰管理插件，提供直观的可视化界面，轻松浏览、搜索、收藏、穿戴和管理海量服饰。

### 📦 核心功能

| 功能 | 说明 |
|:---|:---|
| 🗂️ **分类浏览** | 按服装、头发、面部、身体、饰品五大分类浏览，每个分类下含完整子分类 |
| 🖼️ **缩略图预览** | 网格布局展示服饰缩略图，所见即所得，快速定位心仪服饰 |
| ⭐ **收藏系统** | 一键收藏喜爱的服饰，随时通过筛选快速访问 |
| 👁️ **隐藏系统** | 隐藏不需要的服饰，让列表更清爽，专注真正想要的内容 |
| 📁 **自定义分组** | 创建自定义分组，按自己的方式组织服饰，支持跨分类混合编组 |
| 🔍 **智能搜索** | 按名称、GUID、Zipmod 名搜索，回车即刻刷新结果 |
| 👗 **一键穿戴** | 在角色编辑器中直接从插件穿戴服饰，无需反复切换分类 |
| 🎯 **当前穿着** | 实时显示当前分类正在穿着的服饰信息 |
| 📋 **信息复制** | 一键复制服饰详细信息到剪贴板，方便分享和排查 |

### 🎛️ 个性化设置

| 设置 | 说明 |
|:---|:---|
| 🌐 **多语言** | 支持中文、English、日本語，设置即时生效无需重启 |
| 📄 **每页数量** | 自定义每页显示的服饰数量（10-60） |
| 🔅 **窗口透明度** | 调整非焦点状态下的窗口透明度（20%-100%） |
| 📐 **窗口尺寸** | 自由调整窗口宽度和高度 |

### ⌨️ 快捷操作

- **CTRL + F9** — 显示/隐藏插件窗口

## 📥 安装方法

1. 确保已安装 [BepInEx 5.x](https://github.com/BepInEx/BepInEx) 和 [KKAPI](https://github.com/IllusionMods/KKAPI) 及 [Sideloader](https://github.com/IllusionMods/Sideloader)
2. 将 `HS2_ClothingManager.dll` 放入 `BepInEx/plugins/` 目录
3. 启动游戏，进入角色编辑器后自动加载
